# ForIT Challenge

Hola buenas mi nombre es Tiago y en este proyecto puse mucho de mi para poder llevarlo a cabo use cosas que nunca use pero con ayuda de tutoriales y leer para poder conseguir el objetivo la idea fue construir una aplicación simple pero bien estructurada para gestionar tareas, aplicando buenas prácticas tanto en frontend como en backend.


##  ¿Qué hace la aplicación?

Permite:

- Crear tareas con título y descripción
- Marcar tareas como completadas
- Eliminar tareas
- Visualizar separadas las tareas pendientes y completadas
- Ver un contador de tareas pendientes
- Interfaz moderna y clara

La intención fue mantener la lógica simple pero con una estructura limpia y escalable.


## Tecnologías utilizadas

### Frontend
- React
- TypeScript
- Vite
- TailwindCSS

### Backend
- Node.js
- Express


## Estructura del proyecto

El proyecto está dividido en dos carpetas principales:
/backend
/frontend

- El backend expone una API REST sencilla para manejar las tareas.
- El frontend consume esa API y muestra la información con una interfaz moderna.

## Cómo ejecutar el proyecto primero en Backend

npm run start

## Frontend
npm run dev
